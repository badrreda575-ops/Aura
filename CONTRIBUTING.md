# Contributing to Aura

Thank you for your interest in contributing to Aura! We welcome all contributions, from bug fixes to feature enhancements.

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project.
2. **Create a Branch**: Work on your changes in a dedicated branch.
3. **Commit Your Changes**: Make clear and concise commit messages.
4. **Push to Your Branch**: Push your changes to your fork.
5. **Open a Pull Request**: Submit a PR to the main repository for review.

## Guidelines

- Ensure your code follows the existing style and conventions.
- Provide clear descriptions of your changes in the PR.
- Test your changes locally before submitting.

We value your feedback and contributions!
